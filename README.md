# README

* Ruby version

  use 2.7.4

* System dependencies

  Install postgresql in your OS

* Configuration

  Download this repository and run

  ```
  $ cd articles_app
  $ bundle
  $ yarn install
  ```

* Database creation

  ```
  $ rails db:setup
  $ rails db:migrate
  ```

* Database initialization

  ```
  $ rails db:seed
  ```
